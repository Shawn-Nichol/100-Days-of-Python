# Day 9: Dictionaries, Nesting and the Secret Auction
An overview of my progress, key learnings, and the project built on Day 9 of the 100 Days of Code Python Bootcamp.

<br><br>
### 🛠️ Features
---
* **The Python Dictionary:** Managing data collections using key-value pairs, mastering syntax architecture to create, retrieve, update, and clear values dynamically.
* **Mutating & Iterating Collections:** Looping smoothly through custom dictionaries to isolate keys and extract values for advanced algorithmic grading or calculations.
* **Nesting Complex Structures:** Combining linear and key-value formats seamlessly by embedding lists inside dictionaries, and nesting dictionaries directly inside other dictionaries.
* **Clearing the Console Window:** Leveraging environment-specific operating system commands to dynamically wipe the command-line interface, keeping user screens completely clean and secure.

<br><br>
### 🚀 Day 9 Project: The Secret Auction Program
---
**Description:** A blind-bidding command-line application that allows multiple users to enter private bids sequentially. The program dynamically hides previous inputs via console clearing and evaluates the internal dictionary dataset to declare the highest bidder.
