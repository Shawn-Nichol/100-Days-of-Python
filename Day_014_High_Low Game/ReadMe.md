# Day 14: Higher Lower Game Project
An overview of my progress, key learnings, and the project built on Day 14 of the 100 Days of Code Python Bootcamp.

<br><br>
### 🛠️ Features
---
* **Game Logic & Architecture:** Designing a complete game loop from scratch, tracking user scores dynamically, and handling continuous rounds until a wrong guess occurs.
* **Working with Dictionaries & Lists:** Extracting and displaying formatted data elements from structured list-of-dictionary structures (social media follower datasets).
* **Control Flow & Conditional Logic:** Evaluating complex comparative statements to accurately determine if the player's guess correctly identifies the entity with more followers.
* **Code Reusability with Functions:** Breaking down the codebase cleanly into reusable helper functions for formatting account details and validating player inputs.
* **Dynamic State Management:** Shifting data seamlessly across game rounds, ensuring the correct "Choice B" candidate transitions into the next round's "Choice A" position.

<br><br>
### 🚀 Day 14 Project: Higher Lower Game
---
**Description:** A classic command-line guessing game inspired by the popular website, where players compare the social media follower counts of celebrities, brands, and public figures to build the highest streak possible.
