# Day 1: Working with Variables in Python to Manage Data

An overview of my progress, key learnings, and the project built on Day 1 of the 100 Days of Code Python Bootcamp.
### 🛠️ Features
* **Printing to the Console:** Using the `print()` function to output text data safely onto the screen.
* **String Manipulation:** Working with text strings, including formatting text and combining fragments using the `+` operator (string concatenation).
* **The Input Function:** Capturing user keyboard input from the console using the `input()` function.
* **Python Variables:** Storing dynamic data into named data containers so it can be referenceable later in the program execution.
* **Debugging Practices:** Identifying, troubleshooting, and correcting common text formatting syntax errors and variable reference bugs.

## 🚀 Day 1 Project: Band Name Generator
**Description:** A command-line application that generates a unique, customized band name by combining interactive user input regarding their hometown and a pet's name.

```
print("Welcome to the Band Name Generator.")

bandName = input("What's the name of the city you rew up in?\n")

petName = input("What's your pet's name?\n")

print("Your band name is " + bandName + " " + petName)

```
