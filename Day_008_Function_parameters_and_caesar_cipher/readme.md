# Day 8: Function Parameters & Caesar Cipher
An overview of my progress, key learnings, and the project built on Day 8 of the 100 Days of Code Python Bootcamp.

<br><br>
### 🛠️ Features
---
* **Functions with Inputs:** Dynamically passing arguments into blocks of code via custom parameters, allowing single code blocks to output highly variable, context-driven behavior.
* **Positional Arguments:** Binding data inputs to internal function variables strictly based on the specific sequential order in which they are declared and executed.
* **Keyword Arguments:** Explicitly mapping data inputs directly to target parameters using a `parameter_name = value` syntax to eliminate sequential errors and improve code legibility.
* **Complex Text Manipulation:** Iterating across linear string structures and calculating shifting numerical indices to translate string elements smoothly along the alphabet.
* **Character Demarcation & Filtering:** Implementing defensive validation boundaries inside cipher loops to isolate, preserve, and pass numbers, spaces, and punctuation untouched.

<br><br>
### 🚀 Day 8 Project: Caesar Cipher
---
**Description:** A custom cryptographic command-line application capable of handling complex string encryption and decryption utilizing a shift-key method, featuring modular multi-run loops and robust special character preservation.
