# Day 12: Scope & The Number Guessing Game
An overview of my progress, key learnings, and the project built on Day 12 of the 100 Days of Code Python Bootcamp.

---

## 🛠️ Features
* **Namespaces & Variable Boundaries:** Mastering the distinction between Local Scope (variables confined inside a function) and Global Scope (variables accessible anywhere in the file).
* **Python Constants:** Setting unchangeable static values (like maximum attempts) using the `UPPERCASE` naming convention to represent global variables meant to be read-only.
* **Block Scope Exceptions:** Understanding that unlike other programming languages, Python `if/elif/else` blocks and loops do not create their own isolated variable namespaces.

---

## 🚀 Day 12 Project: The Number Guessing Game
**Description:** An interactive command-line application where the computer selects a secret number between 1 and 100, and the player has to uncover it. The game features distinct difficulty tiers (Easy vs. Hard) that grant differing pools of turn attempts, uses real-time validation to provide directional feedback ("Too high" or "Too low"), and manages turn tracking through a decoupled loop architecture.
